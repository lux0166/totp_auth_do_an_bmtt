# TOTP Demo - Đồ Án Môn Học

Đây là một ứng dụng web demo về xác thực hai yếu tố (2FA) sử dụng Time-Based One-Time Password (TOTP).

## Công nghệ
- Java 17
- Spring Boot
- Maven
- Thymeleaf
- Bootstrap
- Thư viện TOTP: `dev.samstevens.totp`
